AncestorBlaster

High Concept

Shoot words to build together quotes from the Ancestor! (Of Darkest Dungeon Fame). Inspired by EmilyBlaster by Gabrielle Zevin. Play it here.

Genre

Top down arcade shooter.

Platform

Desktop Web only

Story

Abstract Story

Aesthetics

Dark, gritty, stylized in the fashion of the original Darkest Dungeon

Gameplay

The player will be able to control a torch at the bottom of the screen that moves left and right. The player will also be able to fire a flame that goes forward. The bouts of flame will collide with blocks that contain words that together, build the quote. The player will try to match the quote given to them at the beginning of each level.

Documentation

This project was pretty fun to work on, excluding the frustrations with JS as a language, and PIXI's unfamilar style and quirks.

The whole project was pretty straigthforward. Not many changes from the origin design, except that the shooting was changed to just collision at the recommendation of the instructor.

I started by building out the main gameplay loop, making sure the words could be spawned correctly, as well as managing the selection of quotes and garbage words. This constituted about half of development, with the other half mainly taken up by figuring out how to touch up the game with graphics and sound.

I want to highlight the usage of custom fonts in PIXI, which was really annoying to figure out. After a long time spent trying to figure it out on my own, I consulted a youtube video on the subject, and it turns out it was a caching issue, which required loading the font by using it in the HTML itself, and hiding it, so that the JS could use it. That was a headache.

The PIXI docs were super helpful, as documentation tends to be, and using the PIXI debugger extension for Firefox was helpful as well. It helped solve a strange alignment issue regarding trimming and whitespace. I appreciated that it styled itself after Blender, the 3D modelling software.

All requirements are met: I used HTML/CSS/JS + PIXI.JS to build the app. The app is themed to match Darkest Dungeon's theme, and sounds from the game are used as well. HTML and CSS are validated against the validator, and JS code is documented using JSDOC, and follows coding conventions.
